# Java Programming Language

Java is a **high-level, object-oriented programming language** developed by Sun Microsystems (now owned by Oracle). It is widely used for building desktop, web, and mobile applications.

---

## ðŸª„ Key Features

- **Platform Independent:**  
  Java programs run on the *Java Virtual Machine (JVM)*, making them executable on any operating system.

- **Object-Oriented:**  
  Java uses objects and classes to structure programs, improving reusability and maintainability.

- **Automatic Memory Management:**  
  The *Garbage Collector (GC)* automatically removes unused objects from memory.

- **Rich Standard Library:**  
  Java provides thousands of built-in classes for networking, data structures, I/O, and more.
